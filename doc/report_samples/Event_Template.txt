PLAYER KEY
----------------------------------------------------------------------------------------------------
EVM:      Equal Value Match; moves with an evaluation that matches the best engine evaluation
ACPL:     Average Centipawn Loss; sum of total centipawn loss divided by the number of moves
SDCPL:    Standard Deviation Centipawn Loss; standard deviation of centipawn loss values from each move played
Score:    Game Score; measurement of how accurately the game was played, ranges from 0 to 100
ROI:      Raw Outlier Index; normalized Score value where 50 represents the mean and each increment of 5 is one standard deviation from the mean
          An asterisk (*) following an ROI value indicates a situation that deserves extra scrutiny
